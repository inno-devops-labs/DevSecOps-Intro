Key Observations

  • Container/Dependency vulnerabilities dominate (84%) — Trivy (147) + Grype (122) = 269 findings from outdated packages, primarily ReDoS-prone regex libraries (CWE-1333, 29 findings) and path traversal issues.
  • SAST found critical injection flaws — Semgrep identified SQL injection (CWE-89, 6 findings) and prototype pollution (CWE-1321, 6 findings) in Juice Shop's Node.js source code.
  • DAST confirmed runtime misconfigurations — ZAP found 12 findings including missing Content Security Policy, X-Frame-Options, and HSTS headers on the live application.
  • Authentication bypass via JWT none algorithm — Nuclei confirmed a High-severity JWT none-algorithm vulnerability (CWE-347) that allows complete authentication bypass.
  • Deduplication not applied — Cross-tool duplicates exist (e.g., the same CVE appearing in both Trivy and Grype). Real unique count is lower; enabling Dojo deduplication would reduce noise.
